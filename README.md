# Liferay 7


Downloads and unzips the liferay bundle.
```bash 
./gradlew initBundle
```

Assembles the project and deploys it to Liferay.
```bash 
./gradlew deploy
```

Start the server
```bash 
./gradlew server start -b
```

Access the address:
http://localhost:8080/




Assembles the Liferay bundle and zips it up.
```bash 
./gradlew distBundleZip
```
